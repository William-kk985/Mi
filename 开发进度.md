# 开发进度

## 环境

- 仿真容器镜像：`cyberdog_sim:v2026`，来源 `~/Downloads/cyberdog_race2026.tar`
- 启动脚本：`./start_sim.sh --full-build`（全量编译）或 `./start_sim.sh`（只编 cyberdog_locomotion + cyberdog_race）
- 宿主机 `src/` 挂载进容器，代码修改实时同步，但需重新编译才生效

### 常用命令
```bash
# 启动仿真（全量编译）
./start_sim.sh --full-build

# 只编 race 包（改了 stage/motion_ctrl 等）
docker exec -it cyberdog bash -c "cd /home/cyberdog_sim && source /opt/ros/galactic/setup.bash && colcon build --merge-install --packages-select cyberdog_race"

# 编 locomotion 包（改了步高/步态/command_interface 等）
docker exec -it cyberdog bash -c "cd /home/cyberdog_sim && source /opt/ros/galactic/setup.bash && touch src/cyberdog_locomotion/control/src/convex_mpc/convex_mpc_loco_gaits.cpp && colcon build --merge-install --packages-select cyberdog_locomotion"

# 启动比赛控制器
docker exec -it cyberdog bash -c "cd /home/cyberdog_sim && source /opt/ros/galactic/setup.bash && source install/setup.bash && ros2 run cyberdog_race race_controller"
```

### 重要：编译注意事项
- `cyberdog_locomotion` 包含两个关键 so：
  - `libcommon.so`：包含 `command_interface.cpp`（步高命令、gamepad 解析）
  - `libcontrol.so`：包含 `convex_mpc_loco_gaits.cpp`（步高上限、步态参数）
- **`libcontrol.so` 默认不会被增量编译触发**，必须用 `touch` 强制重编
- 编译完必须重启 Control 节点才能加载新 so

## 已完成内容

### 基础框架
- 状态机主控节点 `race_controller`，100Hz 控制循环
- 6 个赛段类（Stage1~6），统一 `StageBase` 接口
- 调试宏体系 `debug_config.hpp`（正式比赛前全部注释掉）

### 步高问题（重要经验）
- 步高控制链路：`Gamepad2Cmd` → `step_height_cmd_` → `WrapRange(min, max)` → `SetHeight()`
- `libcontrol.so` 是 2024 年旧版本，必须用 `touch` 强制重编才能更新
- 最终有效改动：
  - `command_interface.cpp` `Gamepad2Cmd`：`step_height[0] = 0.25`
  - `convex_mpc_loco_gaits.cpp` `SetDefaultParams`：`step_height_min_ = 0.06`（防止贝塞尔轨迹起止段贴地被绊）
  - `cyberdog-ctrl-user-parameters.yaml`：`step_height_max = 0.25`

### 转向符号约定（重要）
- `set_velocity(x, y, yaw)` 中：**yaw 正值 = 左转（yaw 增大），负值 = 右转（yaw 减小）**
- `yaw_err = target_yaw - current_yaw`，`yaw_err > 0` 需左转，发正值
- 之前代码符号写反，已全部修正（stage1 转向、stage2 navigate_to/turn_to/scan）

### MotionCtrl
- 只用 `gamepad_lcmt` 通道，`use_rc=0`
- `set_velocity` 每次发完立刻清零摇杆，依赖 100Hz 控制循环持续发命令
- `stop()` 发零速 locomotion 命令
- Ctrl+C 后 `rclcpp::spin` 退出，main 里发 `stand()` 再 `lie_down()` 停止

### 第一赛段
- 黄线巡线：HSV 分割 + 逐行扫描 + 视觉/IMU 加权融合
  - 直道：IMU 回正为主（权重 0.5~0.8），视觉微调
  - 弯道：纯视觉 PD
  - 低通滤波（α=0.6）减少抖动振荡
- 单边检测时 `lane_both_sides=false`，视觉权重降低到 0.2
- 卡住检测：连续 30 帧位移 < 1cm 触发，加速冲过石板（0.5m/s，持续 40 帧）
- 路口触发：里程计 x ∈ [2.95, 3.2] 触发，转向到 yaw = π/2
- 低头：`set_pitch(-0.26f)`（约 15 度）
